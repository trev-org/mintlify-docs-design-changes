The Microsoft Game Development Kit (GDK) is a unified, comprehensive SDK that gives you everything you need to develop, publish, and run games on Xbox Series X|S, Xbox One, and Windows PC. Built on a shared Game Core programming model, the GDK lets you write cross-platform game code once and target both console and PC with the same set of APIs. Whether you're integrating Xbox services such as achievements, multiplayer, and cloud saves, packaging your title for the Microsoft Store, or squeezing every bit of performance out of Xbox hardware, the GDK is your starting point. From a solo indie developer shipping their first PC title to a large studio shipping a cross-generational console blockbuster, the GDK scales to fit your workflow and engine of choice.

## How to get up and running

Getting from zero to a running GDK project takes just four steps. Follow them in order and you'll have a working development environment — ready to build for Xbox or Windows — in a single session.

<Steps>
  <Step title="Join a developer program">
    Register at [Xbox Developers](https://www.xbox.com/Developers/) to gain access to the GDK, Partner Center, secure downloads, and Xbox developer forums. PC-only titles can use the public GDK, but console development and Xbox services require program membership.
  </Step>
  <Step title="Download and install the GDK">
    Download the latest GDK release from [aka.ms/gdkdl](https://aka.ms/gdkdl) (secure, requires program membership) or clone the [public GDK on GitHub](https://aka.ms/gdk). Run the installer on a 64-bit Windows 10 (May 2019 Update or later) or Windows 11 machine that has Visual Studio 2019 or 2022 installed.
  </Step>
  <Step title="Configure MicrosoftGame.config">
    Every GDK title is identified by a `MicrosoftGame.config` file. Create a product in [Partner Center](https://partner.microsoft.com/dashboard/windows/overview), then copy the Title ID, Identity Name, Publisher, and MSAAppId values into your project's `MicrosoftGame.config` so your game can authenticate with Xbox services and be submitted to the Microsoft Store.
  </Step>
  <Step title="Build and run your game">
    Choose the right Visual Studio project template for your target platform (`Gaming.Desktop.x64` for PC, `Gaming.Xbox.Scarlett.x64` for Xbox Series X|S, or `Gaming.Xbox.XboxOne.x64` for Xbox One), build your project, and deploy. Switch your sandbox to your development sandbox ID and sign in with a test account to validate Xbox services features end-to-end.
  </Step>
</Steps>
