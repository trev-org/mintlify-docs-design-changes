## Xbox Live Context (`xbox_live_context_c`)

| Function | Description |
| --- | --- |
| `XblContextCreateHandle` | Create from a user |
| `XblContextDuplicateHandle` | Duplicate |
| `XblContextCloseHandle` | Release |
| `XblContextGetUser` | Owning user |
| `XblContextGetXboxUserId` | Xuid |

Handle: `XblContextHandle`. Helper struct: `XblGuid`.

## Context Settings (`xbox_live_context_settings_c`)

Get/Set the per-context HTTP retry, timeout, and WebSocket behavior.

`XblContextSettingsGetHttpRetryDelay` / `Set`, `GetHttpTimeoutWindow` / `Set`, `GetLongHttpTimeout` / `Set`, `GetWebsocketTimeoutWindow` / `Set`, `GetUseCrossPlatformQosServers` / `Set`.

## Globals (`xbox_live_global_c`)

| Function | Description |
| --- | --- |
| `XblInitialize` / `XblCleanupAsync` | Library lifecycle (`XblInitArgs`) |
| `XblGetAsyncQueue` | Default queue |
| `XblGetScid` | SCID |
| `XblSetApiType` | Pick API mode (`XblApiType`) |
| `XblSetOverrideConfiguration` / `XblSetOverrideLocale` | Test/dev overrides |
| `XblMemSetFunctions` / `XblMemGetFunctions` | Memory hooks |
| `XblAddServiceCallRoutedHandler` / `Remove` | Inspect every service call |
| `XblDisableAssertsForXboxLiveThrottlingInDevSandboxes` | Quieter dev experience |

Structs: `XblInitArgs`, `XblServiceCallRoutedArgs`. Enums: `XblApiType`, `XblConfigSetting`.

## Platform layer (`platform_c`)

Allows the host to handle local storage on platforms without a default implementation.

`XblLocalStorageSetHandlers` plus the `XblLocalStorage*` callbacks (`ReadHandler`, `WriteHandler`, `ClearHandler`, `ReadComplete`, `WriteComplete`, `ClearComplete`).

Enums: `XblClientOperationResult`, `XblLocalStorageWriteMode`. Handle: `XblClientOperationHandle`.

## Errors (`errors_c`)

`XblGetErrorCondition` and enum `XblErrorCondition` map HRESULT codes to broad conditions (auth, rate-limited, transient, etc.).
