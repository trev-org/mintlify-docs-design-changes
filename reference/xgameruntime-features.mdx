---
title: "XGameRuntimeFeature reference"
description: "Query whether specific Gaming Runtime features are available before using them."
---

After `XGameRuntimeInitialize`, use `XGameRuntimeIsFeatureAvailable` to check whether specific Gaming Runtime features are present.

## XGameRuntimeIsFeatureAvailable

```cpp
bool XGameRuntimeIsFeatureAvailable(XGameRuntimeFeature feature);
```

## XGameRuntimeFeature

```cpp
enum class XGameRuntimeFeature : uint32_t {
    XAccessibility = 0,
    XAppCapture = 1,
    XAsync = 2,
    XAsyncProvider = 3,
    XDisplay = 4,
    XGame = 5,
    XGameInvite = 6,
    XGameSave = 7,
    XGameUI = 8,
    XLauncher = 9,
    XNetworking = 10,
    XPackage = 11,
    XPersistentLocalStorage = 12,
    XSpeechSynthesizer = 13,
    XStore = 14,
    XSystem = 15,
    XTaskQueue = 16,
    XThread = 17,
    XUser = 18,
    XError = 19,
    XGameEvent = 20,
    XGameStreaming = 21,
};
```

## Example

```cpp
bool CheckAsyncTaskQueueFeatures() {
    return XGameRuntimeIsFeatureAvailable(XGameRuntimeFeature::XTaskQueue)
        && XGameRuntimeIsFeatureAvailable(XGameRuntimeFeature::XAsyncProvider)
        && XGameRuntimeIsFeatureAvailable(XGameRuntimeFeature::XAsync);
}
```
