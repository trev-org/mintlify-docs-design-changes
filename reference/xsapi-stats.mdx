## Event-based stats (`user_statistics_c`)

Queries:

- `XblUserStatisticsGetSingleUserStatisticAsync` / `Result` / `ResultSize`
- `XblUserStatisticsGetSingleUserStatisticsAsync` / `Result` / `ResultSize`
- `XblUserStatisticsGetMultipleUserStatisticsAsync` / `Result` / `ResultSize`
- `XblUserStatisticsGetMultipleUserStatisticsForMultipleServiceConfigurationsAsync` / `Result` / `ResultSize`

Tracking and change events:

- `XblUserStatisticsTrackStatistics`, `XblUserStatisticsStopTrackingStatistics`, `XblUserStatisticsStopTrackingUsers`
- `XblUserStatisticsAddStatisticChangedHandler`, `XblUserStatisticsRemoveStatisticChangedHandler`, `XblStatisticChangedHandler`

Types: `XblRequestedStatistics`, `XblServiceConfigurationStatistic`, `XblStatistic`, `XblStatisticChangeEventArgs`, `XblUserStatisticsResult`.

## Title-managed stats (`title_managed_statistics_c`)

- `XblTitleManagedStatsWriteAsync`
- `XblTitleManagedStatsUpdateStatsAsync`
- `XblTitleManagedStatsDeleteStatsAsync`

Struct: `XblTitleManagedStatistic`. Enum: `XblTitleManagedStatType`.
