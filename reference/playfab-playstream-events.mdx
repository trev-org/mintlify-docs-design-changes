PlayStream events are emitted by PlayFab services for every meaningful change. Subscribe via Rules (segment-triggered) or pipe to Insights / external archive. Most events are also available in real-time via the PlayStream debugger.

## Player and identity

Emitted from Authentication, Account management, and Player Profile services.

- `player_created`, `player_logged_in`, `player_authenticated`, `entity_created`, `entity_logged_in`, `entity_deleted`
- `player_added_title`, `player_removed_title`, `logged_in_to_title`
- `player_linked_account`, `player_unlinked_account`, `player_identity_link_added`, `player_identity_link_removed`
- `player_completed_password_reset`, `player_password_reset_link_sent`, `title_initiated_player_password_reset`
- `player_changed_avatar`, `player_displayname_changed`, `player_updated_contact_email`, `player_verified_contact_email`, `player_updated_login_email`
- `player_tag_added`, `player_tag_removed`
- `player_banned`, `player_data_exported`, `player_device_info`, `player_reported_as_abusive`
- `auth_token_validated`, `character_created`
- `title_deleted_master_player`, `title_profile_view_constraints_changed`, `player_custom_properties_changed`

## Inventory and economy

From Economy v2 and the legacy inventory APIs.

- Catalog v1: `title_catalog_updated`, `title_store_updated`
- Catalog v2: `item_created`, `item_updated`, `item_deleted`, `item_publish_requested`, `item_moderation_state_changed`, `item_reported`, `review_reported`, `reviews_takedown_requested`
- Inventory: `items_added`, `items_subtracted`, `items_updated`, `items_deleted`, `items_expired`, `items_purchased`, `items_redeemed`, `items_transferred`, `items_clawback_completed`, `items_clawback_reversal_completed`, `marketplace_transaction_redeemed`
- Character / virtual currency: `character_consumed_item`, `character_inventory_item_added`, `character_vc_item_purchased`, `character_virtual_currency_balance_changed`
- Player VC: `player_consumed_item`, `player_inventory_item_added`, `player_vc_item_purchased`, `player_virtual_currency_balance_changed`, `entity_virtual_currency_balances_changed`
- Purchasing: `player_started_purchase`, `player_paid_for_purchase`
- Player set / profile property: `player_set_profile_property`
