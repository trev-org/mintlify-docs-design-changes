## Get and set presence

| Function | Description |
| --- | --- |
| `XblPresenceSetPresenceAsync` | Set your title's presence for the local user |
| `XblPresenceGetPresenceAsync` / `Result` | Read one user's record |
| `XblPresenceGetPresenceForMultipleUsersAsync` / `Result` / `ResultCount` | Bulk query |
| `XblPresenceGetPresenceForSocialGroupAsync` / `Result` / `ResultCount` | Query a social group |

## Records

| Function | Description |
| --- | --- |
| `XblPresenceRecordGetXuid` | Owner xuid |
| `XblPresenceRecordGetUserState` | High-level state |
| `XblPresenceRecordGetDeviceRecords` | Per-device records |
| `XblPresenceRecordDuplicateHandle` / `CloseHandle` | Lifetime |

## Change handlers

`XblPresenceAddDevicePresenceChangedHandler` / `Remove`, `XblPresenceDevicePresenceChangedHandler`, `XblPresenceAddTitlePresenceChangedHandler` / `Remove`, `XblPresenceTitlePresenceChangedHandler`.

Tracking helpers: `XblPresenceTrackUsers`, `XblPresenceStopTrackingUsers`, `XblPresenceTrackAdditionalTitles`, `XblPresenceStopTrackingAdditionalTitles`.

## Types

Structs: `XblPresenceBroadcastRecord`, `XblPresenceDeviceRecord`, `XblPresenceQueryFilters`, `XblPresenceRichPresenceIds`, `XblPresenceTitleRecord`.

Enums: `XblPresenceBroadcastProvider`, `XblPresenceDetailLevel`, `XblPresenceDeviceType`, `XblPresenceMediaIdType`, `XblPresenceTitleState`, `XblPresenceTitleViewState`, `XblPresenceUserState`.

Handle: `XblPresenceRecordHandle`.
