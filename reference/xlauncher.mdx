`XLauncher` is the GDK's gate for opening URIs (`https:`, `ms-windows-store:`, partner protocols) from within a game.

## Functions

| Function | Description |
| --- | --- |
| `XLaunchUri` | Launch a URI — the system may prompt the user before opening |
