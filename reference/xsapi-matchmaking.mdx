## Tickets

| Function | Description |
| --- | --- |
| `XblMatchmakingCreateMatchTicketAsync` / `Result` | Submit a ticket to a hopper |
| `XblMatchmakingDeleteMatchTicketAsync` | Cancel a ticket |
| `XblMatchmakingGetMatchTicketDetailsAsync` / `Result` / `ResultSize` | Query a ticket |

## Hoppers

| Function | Description |
| --- | --- |
| `XblMatchmakingGetHopperStatisticsAsync` / `Result` / `ResultSize` | Wait times, players in queue |

## Types

Structs: `XblCreateMatchTicketResponse`, `XblHopperStatisticsResponse`, `XblMatchTicketDetailsResponse`.

Enums: `XblPreserveSessionMode`, `XblTicketStatus`.
