`XGameEvent` is the lightweight API for writing structured game events that drive Xbox event-based achievements and stats, plus PlayFab/Insights analytics.

## Functions

| Function | Description |
| --- | --- |
| `XGameEventWrite` | Write a game event with a payload to the Events service |

Event schemas are configured in Partner Center under **Xbox services → Gameplay settings → Events**.
