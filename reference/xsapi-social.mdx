## Relationships

| Function | Description |
| --- | --- |
| `XblSocialGetSocialRelationshipsAsync` / `Result` | Fetch the local user's friends/follows |
| `XblSocialRelationshipResultGetRelationships` | Read items |
| `XblSocialRelationshipResultGetTotalCount` / `HasNext` / `GetNextAsync` / `GetNextResult` | Paginate |
| `XblSocialRelationshipResultDuplicateHandle` / `CloseHandle` | Lifetime |

## Change events

`XblSocialAddSocialRelationshipChangedHandler`, `XblSocialRemoveSocialRelationshipChangedHandler`.

## Reputation feedback

`XblSocialSubmitReputationFeedbackAsync`, `XblSocialSubmitBatchReputationFeedbackAsync`.

## Types

Structs: `XblReputationFeedbackItem`, `XblSocialRelationship`, `XblSocialRelationshipChangeEventArgs`. Handle: `XblSocialRelationshipResultHandle`. Enums: `XblReputationFeedbackType`, `XblSocialNotificationType`, `XblSocialRelationshipFilter`.
