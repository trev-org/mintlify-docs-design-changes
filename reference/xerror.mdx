`XError` lets you install a process-wide callback that's invoked when the GDK detects a critical error — useful for centralized telemetry, crash reporting, and last-chance logging.

## Functions

| Function | Description |
| --- | --- |
| `XErrorSetCallback` | Install the error callback |
| `XErrorSetOptions` | Configure callback behavior (`XErrorOptions` flags) |
| `XErrorCallback` | Callback signature |

## Pattern

```cpp
void CALLBACK MyErrorCallback(HRESULT hr, const char* message, void* context) {
  // Log to telemetry, write a minidump hint, etc.
}

XErrorSetCallback(MyErrorCallback, nullptr);
```
