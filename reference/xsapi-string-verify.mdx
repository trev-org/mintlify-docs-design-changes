Use `string_verify_c` for guild names, clan tags, custom statuses, in-game chat, and any other user-authored strings you display or persist.

## Functions

| Function | Description |
| --- | --- |
| `XblStringVerifyStringAsync` / `Result` / `ResultSize` | One string |
| `XblStringVerifyStringsAsync` / `Result` / `ResultSize` | Batch |

## Types

Struct: `XblVerifyStringResult`. Enum: `XblVerifyStringResultCode`.
