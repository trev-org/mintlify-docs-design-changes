The `multiplayer_c` module is the raw MPSD API surface. Use it directly when you need full control of session JSON. For higher-level orchestration, prefer [Multiplayer Manager](/reference/xsapi-multiplayer-manager).

## Sessions — read

| Function | Description |
| --- | --- |
| `XblMultiplayerGetSessionAsync` / `Result` | Read a session by reference |
| `XblMultiplayerGetSessionByHandleAsync` / `Result` | Read a session by handle |
| `XblMultiplayerQuerySessionsAsync` / `Result` / `ResultCount` | Query sessions |

## Sessions — write

| Function | Description |
| --- | --- |
| `XblMultiplayerWriteSessionAsync` / `Result` | Write by session reference |
| `XblMultiplayerWriteSessionByHandleAsync` / `Result` | Write by handle |

## Search handles (advertised sessions)

Create / delete: `XblMultiplayerCreateSearchHandleAsync` / `Result`, `XblMultiplayerDeleteSearchHandleAsync`. Query: `XblMultiplayerGetSearchHandlesAsync` / `Result` / `ResultCount`.

Handle accessors: `XblMultiplayerSearchHandleGetId`, `GetSessionReference`, `GetCreationTime`, `GetCustomSessionPropertiesJson`, `GetJoinRestriction`, `GetMemberCounts`, `GetNumberAttributes`, `GetSessionClosed`, `GetSessionOwnerXuids`, `GetStringAttributes`, `GetTags`, `GetVisibility`. Lifetime: `XblMultiplayerSearchHandleDuplicateHandle`, `CloseHandle`.

## Activity, invites, and transfer

Activity: `XblMultiplayerSetActivityAsync`, `XblMultiplayerClearActivityAsync`, `XblMultiplayerGetActivitiesForUsersAsync` / `Result` / `ResultCount`, `XblMultiplayerGetActivitiesForSocialGroupAsync` / `Result` / `ResultCount`, plus the `WithProperties` variants.

Invites: `XblMultiplayerSendInvitesAsync` / `Result`.

Transfer handles: `XblMultiplayerSetTransferHandleAsync` / `Result`.

Secure device address helper: `XblFormatSecureDeviceAddress` (`XblFormattedSecureDeviceAddress`).

## Subscriptions

Real-time: `XblMultiplayerSetSubscriptionsEnabled`, `XblMultiplayerSubscriptionsEnabled`.

Handlers: `XblMultiplayerAddSessionChangedHandler` / `Remove`, `XblMultiplayerSessionChangedHandler`, `XblMultiplayerAddConnectionIdChangedHandler` / `Remove`, `XblMultiplayerConnectionIdChangedHandler`, `XblMultiplayerAddSubscriptionLostHandler` / `Remove`, `XblMultiplayerSessionSubscriptionLostHandler`.
