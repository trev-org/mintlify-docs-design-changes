## Functions

| Function | Description |
| --- | --- |
| `XblProfileGetUserProfileAsync` / `Result` | One user |
| `XblProfileGetUserProfilesAsync` / `Result` / `ResultCount` | Many users |
| `XblProfileGetUserProfilesForSocialGroupAsync` / `Result` / `ResultCount` | Filter by social group |

## Structure

`XblUserProfile` — gamertag, real name, app display name, profile pic URL, account tier, gender.
