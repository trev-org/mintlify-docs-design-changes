`XGame` is the small surface for cross-title transitions and crash recovery.

## Functions

| Function | Description |
| --- | --- |
| `XGameGetXboxTitleId` | Get the current title's Xbox Title ID |
| `XLaunchNewGame` | Launch a different game on the device (closes the current game) |
| `XLaunchRestartOnCrash` | Configure automatic restart behavior on crash |
