## Runtime

| Code | Value | Meaning |
| --- | --- | --- |
| `E_GAME_MISSING_GAME_CONFIG` | `0x87E5001F` | Missing required `MicrosoftGame.config` |
| `E_GAMERUNTIME_NOT_INITIALIZED` | `0x89240100` | Game Runtime not initialized |
| `E_GAMERUNTIME_DLL_NOT_FOUND` | `0x89240101` | Game Runtime DLL not found |
| `E_GAMERUNTIME_VERSION_MISMATCH` | `0x89240102` | DLL doesn’t support this GDK version |
| `E_GAMERUNTIME_WINDOW_NOT_FOREGROUND` | `0x89240103` | Window must be foreground |
| `E_GAMERUNTIME_SUSPENDED` | `0x89240104` | Runtime is suspended |
