# Using Performance Investigator (PIX) to profile Windows titles

Feedback

Summarize this article for me

The version of PIX that ships in the Gaming eXtensions Development Kit (GXDK) supports the profiling of titles on Xbox One Development Kits. If you're building a version of your title for the PC, you need to install [PIX for Windows](https://blogs.msdn.microsoft.com/pix/) to profile.

Note

PIX for Windows can only be used on Windows 10 and later and supports only DirectX 12.

PIX for Windows and PIX for Xbox can be installed side-by-side on the same development PC.

For more information about installing and using PIX for Windows, see the [PIX online documentation](https://blogs.msdn.microsoft.com/pix/documentation/).
