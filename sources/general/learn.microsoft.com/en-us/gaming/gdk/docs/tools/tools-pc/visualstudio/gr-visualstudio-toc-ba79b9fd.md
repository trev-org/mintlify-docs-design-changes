# Visual Studio (contents)

Feedback

Summarize this article for me

Use the Visual Studio Integrated Development Environment (IDE) to create, build, deploy, and debug Microsoft Game Development Kit (GDK) on PC apps. The GDK supports Visual Studio 2019 (16.11) and Visual Studio 2022. For GDK development, use Visual Studio 2019 version 16.11 or the latest Visual Studio 2022 update.

Note

If you install a new edition of Visual Studio after installing the GDK, you can repair the GDK to integrate with the newly installed edition of Visual Studio. This integration isn't needed for servicing updates.
