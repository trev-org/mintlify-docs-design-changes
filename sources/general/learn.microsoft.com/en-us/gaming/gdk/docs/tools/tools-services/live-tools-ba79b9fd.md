# Development tools for Xbox services

Feedback

Summarize this article for me

Various tools in the GDK can help you develop for Xbox services.

To get these Xbox services development tools, launch a GDK console or call the tools directly in the GDK bin folder.

Many of these tools are also available using this [aka.ms link](https://aka.ms/xbltools) or in the [Xbox Service Developer Tools GitHub](https://github.com/Microsoft/xbox-live-developer-tools) repo.

To create your own custom tooling, you can use Microsoft.Xbox.Service.DevTools.dll in the GDK or the [Dev Tools library source code](https://github.com/microsoft/xbox-live-developer-tools/tree/main/Microsoft.Xbox.Service.DevTools).

## Xbox service tools

[Config as Source tools (XblConfig.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-config-as-source?view=gdk-2604) 
Enables advanced users to integrate their tools with Xbox configuration services.

[Xbox Connected Storage tool (XblConnectedStorage.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-connected-storage-tool?view=gdk-2604) 
The connected Storage tool enables developers to download save data from their titles.

[Global Storage tool (GlobalStorage.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-global-storage-tool?view=gdk-2604) 
GlobalStorage.exe is used to manage title global storage in test sandboxes, before publish to RETAIL.

[Multiplayer Session History Viewer (MultiplayerSessionHistoryViewer.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-mp-session-history-viewer?view=gdk-2604) 
Shows a historical timeline of all changes over a multiplayer session document's history.

[Player Data Reset tool (XblPlayerDataReset.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-player-data-reset?view=gdk-2604) 
Resets a player's data in test sandboxes.

[Test Account tool (XblTestAccountGui.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-xbox-live-account-tool?view=gdk-2604) 
Sets up existing test accounts for testing game scenarios.

[Developer Account tool (XblDevAccount.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-dev-account-tool?view=gdk-2604) 
Manages authentication of a developer account and saves the credentials to be used across other Xbox services executables.

[PC Sandbox Switcher (XblPCSandbox.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-pc-sandbox-switcher?view=gdk-2604) 
Command-line tool that helps manage the Xbox Live sandbox on PC.

[Trace Analyzer (XblTraceAnalyzer.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-trace-analyzer?view=gdk-2604) 
Reviews the service calls made by a title.

[Using Fiddler to inspect web service calls](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-fiddler-inspect-web-calls?view=gdk-2604) 
Using Fiddler to log and troubleshoot Xbox Live service calls.

[Store Development Entitlement Tool (DevEntitlementTool.exe)](https://learn.microsoft.com/en-us/gaming/gdk/docs/tools/tools-services/live-dev-entitlements-tool?view=gdk-2604) Command-line tool to query, add, and remove Microsoft Store entitlements to/from Xbox test accounts in sandbox.
