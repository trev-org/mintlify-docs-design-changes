# Microsoft Store Service APIs

Feedback

Summarize this article for me

This section lists technical reference articles for Microsoft Store service APIs used by game services. For planning and implementation guidance, see [Manage products from your services](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/service-to-service-nav?view=gdk-2604).

## In this section

[Fine Grained Rate Limiting for Microsoft Store APIs](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/xstore-fgrl?view=gdk-2604) 
Describes calling patterns and fine-grained rate limiting for Microsoft Store service calls.

[collections.mp.microsoft.com/v9.0/collections/publisherQuery](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v9-query-for-products?view=gdk-2604) 
Describes how to use publisherQuery (v9) to query owned products, including Xbox Game Pass status.

[collections.mp.microsoft.com/v8.0/collections/b2bLicensePreview](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-query-for-products?view=gdk-2604) 
Describes how to use b2bLicensePreview (v8) to query owned products from your service.

[collections.mp.microsoft.com/v8.0/collections/consume](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-consume?view=gdk-2604) 
Describes how to fulfill Developer-managed consumables or remove quantity from Store-managed consumables.

[purchase.mp.microsoft.com/v8.0/b2b/orders/query](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-clawbackv1?view=gdk-2604) 
Describes the legacy query for refunded/revoked consumable purchases.

[purchase.mp.microsoft.com/v8.0/b2b/clawback/sastoken](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-clawbackv2-sastoken?view=gdk-2604) 
Generates an SAS token to authenticate with your Clawback event message queue.

[purchase.mp.microsoft.com/v7.0/purchases/grant](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-grant?view=gdk-2604) 
Describes how to grant free app or add-on products to a user from your service.

[purchase.mp.microsoft.com/v8.0/b2b/recurrences/query](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-recurrence-query?view=gdk-2604) 
Describes how to query a user's subscription products.

[purchase.mp.microsoft.com/v8.0/b2b/recurrences/{recurrenceId}/change](https://learn.microsoft.com/en-us/gaming/gdk/docs/store/commerce/service-to-service/microsoft-store-apis/xstore-v8-recurrence-change?view=gdk-2604) 
Describes how to change a user's subscription billing state.
