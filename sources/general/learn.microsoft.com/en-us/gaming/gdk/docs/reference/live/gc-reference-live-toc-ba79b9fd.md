# Xbox services API reference

Feedback

Summarize this article for me

Xbox services API reference material for the Microsoft Game Development Kit (GDK).

## In this section

[httpClient](https://learn.microsoft.com/en-us/gaming/gdk/docs/reference/live/httpclient/atoc-httpclient?view=gdk-2604) 
HTTPClient API reference material for the Microsoft Game Development Kit (GDK).

[XSAPI](https://learn.microsoft.com/en-us/gaming/gdk/docs/reference/live/xsapi-c/atoc-xsapi-c?view=gdk-2604) 
XSAPI reference material for the Microsoft Game Development Kit (GDK).

[Xbox Live HRESULT Error Codes](https://learn.microsoft.com/en-us/gaming/gdk/docs/reference/live/xbl-hresult-error-codes?view=gdk-2604)

[REST](https://learn.microsoft.com/en-us/gaming/gdk/docs/reference/live/rest/atoc-xboxlivews-reference?view=gdk-2604) 
Services for managing games and player information. Use the **Xbox.Services** API (XSAPI)whenever possible. Use the RESTful interface for newer methods that have not been added to **Xbox.Services** yet.
