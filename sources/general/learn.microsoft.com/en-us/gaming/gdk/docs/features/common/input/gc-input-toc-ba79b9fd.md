# Input

Feedback

Summarize this article for me

Physical input is the gamer's window into the game. It's where their intent ends and the game begins. It's vital that each game's input is seamless, responsive, and catered to the experience.

[Overview of GameInput](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-overview?view=gdk-2604) Provides an overview of the GameInput API. This is the place to start.

[GameInput fundamentals](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-fundamentals?view=gdk-2604) Describes the GameInput programming model and design.

[GameInput readings](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-readings?view=gdk-2604) Provides details about GameInput readings.

[GameInput devices](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-devices?view=gdk-2604) Provides details about GameInput devices.

[GameInput Frequently Asked Questions](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-faq?view=gdk-2604) Answers to frequently asked questions about the GameInput API.

[GameInput API versioning](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-versioning?view=gdk-2604) Explains how changes to the GameInput API impact your game.

[GameInput for PC with NuGet](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/input-nuget?view=gdk-2604) Explains how to use GameInput for PC titles using the NuGet package.

[GameInput advanced topics](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/advanced/input-advanced-topics?view=gdk-2604) Covers a variety of topics about advanced use cases.

[GameInput for hardware developers](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/hardware/input-hardware-topics?view=gdk-2604) Explains how to build devices which work seamlessly with GameInput.

[Virtual Keyboard support](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/overviews/virtualkeyboardsupport?view=gdk-2604) Explains how to bring up the system-provided virtual keyboard in your application.
