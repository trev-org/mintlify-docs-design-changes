# Packaging

Feedback

Summarize this article for me

This section describes the packaging features of the Microsoft Game Development Kit (GDK).

## In this section

[Overview of packaging](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/overviews/packaging?view=gdk-2604) 
Provides an overview of packaging.

[Getting Started Packaging for Xbox Consoles](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/overviews/packaging-getting-started-for-console?view=gdk-2604) 
Describes getting started with packaging for Xbox consoles.

[Getting started with packaging for PC](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/overviews/packaging-getting-started-for-pc?view=gdk-2604) 
Describes getting started with packaging for PC.

[Streaming Installation and Intelligent Delivery: an overview](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/overviews/streaming_install-intelligent_delivery?view=gdk-2604) 
Provides an overview of the Streaming Installation and Intelligent Delivery features, including how title data is organized into chunks to facilitate fast and flexible installation.

[Common packaging tasks and workflow overview](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/title-packaging-streaming-install-testing?view=gdk-2604) 
Describes how to create packages for streaming installation for local iteration, performance validation, pre-submission testing, and final submission. It also describes the steps run by the MakePkg tool during the packaging process.

[Intelligent Delivery features](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/overviews/intelligentdelivery?view=gdk-2604) 
Describes how to use Intelligent Delivery to install only a subset of content.

[Tools for Streaming Installation](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/deployment/atoc-deployment-tools?view=gdk-2604) 
This section presents the tools and schemas that are related to the streaming installation of titles.

[Automating package uploading](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/package-uploader?view=gdk-2604) 
Describes how to automate uploading of packages to Partner Center.
