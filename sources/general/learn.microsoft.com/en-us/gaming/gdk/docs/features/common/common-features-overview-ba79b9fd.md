# Common GDK features

Feedback

Summarize this article for me

The features described in this section are common to all supported by the GDK for all platforms, including Console and PC.

## Asynchronous programming model

The GDK uses a new programming model for handling asynchronous calls.

[Asynchronous Programming Overview](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/async/async-toc?view=gdk-2604)

## User model

The user model is how the GDK tracks the identify and activity of someone playing a game.

[Users](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/user-toc?view=gdk-2604)

## Packaging

These topics cover how to create a GDK package to upload your game to the store:

[Packaging](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/packaging/gc-packaging-toc?view=gdk-2604)

## Game configuration

The topics in this section describe how to configure the MicrosoftGame.config file to define settings for your game.

[MicrosoftGame.config](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/game-config/microsoftgameconfig-toc?view=gdk-2604)

## Game saves

You can use the Game Saves feature to sync game save status across multiple devices for the same user.

[Game saves (contents)](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/game-save/game-saves-toc?view=gdk-2604)

## Game streaming

This section contains topics to help your game take advantage of Xbox Game Streaming and Xbox Remote Play.

[Introduction to game streaming](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/game-streaming/game-streaming-overview?view=gdk-2604)

## Game input

Game input is how the player interacts with the game by using various input devices, such as controllers, keyboard and mouse, or other peripherals.

[Input](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/input/gc-input-toc?view=gdk-2604)

## Lighting

The GDK provides API support for some hardware that allows programs to dynamically alter the lighting of that hardware.

[Lighting API Overview](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/lighting/gc-lighting-toc?view=gdk-2604)
