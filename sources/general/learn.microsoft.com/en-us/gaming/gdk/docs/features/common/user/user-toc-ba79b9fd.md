# Users

Feedback

Summarize this article for me

This section describes the Microsoft Game Development Kit (GDK) concept of a user, how a user is represented as an `XUser`, and the relationship of a user to input devices.

## In this section

[User identity and XUser](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/player-identity-xuser?view=gdk-2604) 
Describes how Xbox One manages the identity of the users interacting with the game with an `XUser` object. Each `XUser` instance represents a user who is signed in to the game.

[GDK User Models (NDA topic)Authorization required](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/gamecore-user-models?view=gdk-2604) 
Describes the two users models available in the GDK

[Opting into Default User Required (NDA topic)Authorization required](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/users-opting-into-simplified-model?view=gdk-2604) 
Describes how to opt into ensuring the game has a default user at launch

[Establishing the Default User for the Game (NDA topic)Authorization required](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/users-establishing-default-user?view=gdk-2604) 
Describes how to establish the default user in the game.

[Changing the Default User for the Game (NDA topic)Authorization required](https://learn.microsoft.com/en-us/gaming/gdk/docs/features/common/user/users-changing-default-user?view=gdk-2604)
