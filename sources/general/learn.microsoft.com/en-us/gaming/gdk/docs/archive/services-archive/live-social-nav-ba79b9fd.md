## In this section

[Overview of Social features](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/live-social-overview?view=gdk-2604) 
Xbox services Social features provide a gaming-oriented social network that includes a user's profile, gamerscore, achievements, and activity feed.

[Social Manager](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/social-manager/live-social-manager-nav?view=gdk-2604) 
Provides information about how the Xbox services Social Manager API simplifies keeping track of online friends and their gaming activity.

[People System (Friends List)](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/people-system/live-people-system-nav?view=gdk-2604) 
Describes how the People System includes the Friends List and manages relationships between players, like only showing a player's real name to close friends.

[Rich Presence](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/presence/live-presence-nav?view=gdk-2604) 
Provides details about Rich Presence, which shows what a player is doing and where they are in the game.

[Localized strings](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/localized-strings/live-localized-strings-nav?view=gdk-2604) 
Provides information about configuring localized strings in all the languages that are associated with a game.

[Reputation](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/community/reputation/live-reputation-nav?view=gdk-2604) 
The Reputation service encourages positive gameplay.
