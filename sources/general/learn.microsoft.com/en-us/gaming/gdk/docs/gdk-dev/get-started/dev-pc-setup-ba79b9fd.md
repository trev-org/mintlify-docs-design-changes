# Set up your developer environment

Feedback

Summarize this article for me

This section outlines the setup steps that are required for developing both Xbox and PC games using the GDK.

You should perform the steps in each page in the order the pages appear.

After going through the setup steps in this section, follow the steps in [Get started with the Microsoft Game Development Kit](https://learn.microsoft.com/en-us/gaming/gdk/docs/gdk-dev/get-started/get-started-home?view=gdk-2604).
