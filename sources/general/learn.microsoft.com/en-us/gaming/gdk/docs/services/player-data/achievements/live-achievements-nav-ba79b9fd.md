# User achievements systems

Feedback

Summarize this article for me

This section describes the various ways to track and manage user's achievements in a title.

## In this section

[Event-based vs. title-managed Achievements](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/player-data/achievements/live-achievements-eb-vs-tm?view=gdk-2604) 
Describes the differences between event-based and title-managed Achievements.

[Event-based achievements](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/player-data/achievements/event-based/live-achievements-eb-nav?view=gdk-2604) 
Describes the event-based Achievements service that's based on telemetry and user statistics.

[Title-managed achievements](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/player-data/achievements/title-managed/live-achievements-tm-nav?view=gdk-2604) 
Provides an overview of title-managed Achievements, its concepts, and how to configure it. Example code is also provided.

[Achievements Manager](https://learn.microsoft.com/en-us/gaming/gdk/docs/services/player-data/achievements/achievements-manager/live-achievements-manager-nav?view=gdk-2604) 
The Xbox Live Achievements Manager API simplifies keeping track of, and updating the state of an Xbox Live user's achievements.
