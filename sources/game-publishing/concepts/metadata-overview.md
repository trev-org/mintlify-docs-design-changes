# Metadata overview

The **Gaming Metadata** module is designed to empower you to connect with players by directly showcasing specific features of your content to improve discoverability of content within our storefronts. Players will be able to browse and explore content based on their interests and needs that match the specific features you have chosen to highlight within this module.

This module enables you to configure the tags that appear on your game's Product Details Page (PDP) in the Microsoft Store. All sections are optional and can be updated at any time.

## Configuration guidance

Practical information to help you configure the metadata module.

### Accessibility features

Configure your game's accessibility feature tags to inform players about the ways in which your game has been made accessible. Accessibility feature tag declarations are subject to validation by Microsoft. However, you won't be blocked from publishing while this validation takes place. If the validation result indicates that changes are needed in your feature declarations, you'll need to update these before you publish updates for your product.

To be sure that you're configuring accessibility feature tags correctly, review the [Accessibility Feature Requirements](https://aka.ms/afts).

If you're interested in learning more about gaming accessibility, review the [Xbox Accessibility Guidelines](https://aka.ms/xags) or consider submitting your game for the [Microsoft Gaming Accessibility Testing Service](https://aka.ms/mgats).

### Supported languages

Configure your game's supported language information to inform players about the languages that your game supports. Supported languages can be defined independently for user interface, voice, and subtitles.

If you're distributing market-specific game packages, and the supported languages vary depending on the package available in each market, check the box for **Language support may vary depending on Store region and initial download** to indicate this on your PDP.

### Game Catalog Extension ID

> [!Warning]
> It's critical that you don't enter any information in this field unless directed to do so by your Microsoft contacts. Incorrect information in this field will result in game installation issues.

If your game is part of the Game Catalog Extension Program, enter your extension ID here.

### External content

Configure which external content you'd like to display on your PDP. This content might not be visible on all Store endpoints.
