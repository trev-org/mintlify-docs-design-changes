# Create and reserve your game name

[!INCLUDE [reminder](../includes/managed-creators-only-feature.md)]

All apps and games on the Microsoft Store must have a unique name. The first step toward publishing your product in the Store is to reserve the name you'd like to use. You can reserve your game's name as soon as you create the product. This allows you to reserve a codename during development and the final name for the product after it's confirmed. The name reservation will be removed if the product isn't published to a sandbox within three months of reserving the name. 

To reserve a name during product creation, use the following steps.
 1. Sign in to [Partner Center](https://partner.microsoft.com).
 2. Go to the Partner Center **Apps and games** page.
 3. Select **New product**.
 4. Select your **product's type**, for example **Game** or **Game Demo**.
 5. Enter the name you want to use, and then select **Check availability**. If the name is available, you'll see a green check mark. If the name is already in use, you'll see a message to indicate that.
 6. After you've selected an available name to reserve, select **Reserve product name**.

We recommend reserving your name as soon as possible to ensure it will be available when you're ready to publish.

* For all **Game** products, reserve the final product name as soon as you can, including any localized title names if applicable.
* For **Game Preview** products, reserve both the product name and the product name with the (Game Preview) suffix, including any localized title names if applicable.

If the name you're trying to reserve isn't available, contact your Microsoft account team or ID@Xbox contacts.

> [!NOTE]
> The initial name reservation is used for the display name in the Partner Center dashboard. This also appears in the **Identity Details** of the product which might be visible to players after you publish it to the RETAIL sandbox. These **Identity Details** can't be changed later.

## Reserve additional game names

After you’ve created your product and reserved a name, you might need to reserve other names for the game; for example, if the name of your game is changing, you used a temporary name during product creation, or you need to reserve the name in another language. To reserve additional names, use the following steps.

 1. Sign in to [Partner Center](https://partner.microsoft.com).
 2. Go to the Partner Center **Apps and games** page, and then select the product you want to reserve a new name for.
 3. On the product page's navigation, select **Manage product names**.
 4. Enter the name you want to use, and then select **Check availability**. If the name is available, you'll see a green check mark. If the name is already in use, you'll see a message to indicate that.
 5. After you've selected an available name to reserve, select **Reserve product name**.

## Delete a reserved name

If you no longer need a name that you’ve reserved for your product, you can delete that name. Deleting it might make it available for other users to reserve. To delete a name, use the following steps.

 1. Sign in to [Partner Center](https://partner.microsoft.com).
 2. Go to the Partner Center **Apps and games** page, and then select the product you want to reserve a new name for.
 3. On the product page's navigation, select **Manage product names**.
 4. Go to the name that you want to delete, and then select **Delete**. 

## Set a dashboard name

For products that are displayed in the **Apps and games** dashboard, you can update the name that’s displayed for product types that support name reservations; for example, Game and Game Demo. To do so, go to the **Manage product names** page for your product, and then select **Set as dashboard name** for any reserved product name. However, the name that’s displayed in the **Apps and games ** dashboard can't be updated for product types that don't support name reservations; for example, Game Bundles.
