# Licensed Test Vendors  

Updated March 27, 2025  

The following third-party test vendors have been licensed by Microsoft to work with Xbox development tools, including the Microsoft Game Development Kit (GDK) software and Xbox development hardware. Microsoft provides this list as a reference for publishers and developers seeking test vendors who are licensed to test Xbox console and Xbox PC titles.

Please contact vendors directly to inquire about the services they offer. If a test vendor requests confirmation of your GDK license status, please contact your Microsoft representative. [Who are your Microsoft contacts?](../../game-publishing/resources/managed-support/overview-microsoft-representatives-and-contacts.md)

Microsoft only confirms the GDK license status for these vendors as of the publication date above. You remain responsible for any fees associated with using these testing services, and Microsoft does not make any warranties or guarantees regarding the test vendors listed, except for confirmation of their GDK license status. Working with a licensed test vendor does not change the certification process applied to titles submitted to Microsoft.
