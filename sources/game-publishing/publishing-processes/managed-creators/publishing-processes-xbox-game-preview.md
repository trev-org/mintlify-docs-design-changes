# Xbox Game Preview

[!INCLUDE [reminder](../../includes/managed-creators-only-feature.md)]

Xbox Game Preview is a way for fans to preview and purchase work-in-progress digital games, participate in the development process, and help Creators make Xbox games the best they can be. Game Preview games are still in development and might be missing content, contain bugs, or need extra balancing work that comes later via Content Updates.

Game Preview is available for both Xbox console and PC games. 

## Applying for Game Preview

Before applying for Game Preview, you must have an approved [Game Concept](publishing-processes-game-concepts.md) for your game. After your game receives concept approval, review the Xbox Game Preview program policy documentation in the [Xbox Publisher Guide](https://aka.ms/xboxpublisherguide). The policy document explains how to apply for Game Preview and includes the application form you must complete and submit.

The Xbox Game Preview policy documentation also contains other important information that you need to be aware of when participating in Game Preview. Review the policy documentation in full and speak to your Microsoft contacts if you have any questions.

## Game Preview metadata and assets

After successfully registering for Game Preview, download the localized disclaimers from the [Xbox Publisher Guide](https://aka.ms/xboxpublisherguide) and the branding templates from the [Xbox Brand Assets](https://developer.microsoft.com/games/resources/xbox-brand-assets/) portal. These assets are required as you set up your game's [Store listings](../../concepts/store-listing.md) for display in the Microsoft Store.

## Xbox Requirements for Game Preview

Game Preview games are required to pass [Xbox Certification](../../concepts/certification/certification-guide.md) before they can be released. While your game is participating in Game Preview, only the [Xbox Requirements for Xbox Game Preview](/gaming/gdk/_content/gc/policies/certification-requirements-beta-game-preview#game-preview) apply and are tested during Certification.

Games that release in Game Preview can become eligible for [Xbox Cert Bypass](../../concepts/certification/cert-bypass.md). If you make significant changes to your game during Game Preview—such as adding multiplayer features to a game that was previously single-player—you might be required to opt in to full Certification testing. Speak to your Microsoft contacts if you're not sure whether your update is eligible for Cert Bypass.

## Leaving Game Preview

When you're ready for your game to leave Game Preview, contact your Microsoft representatives for guidance on required product changes. To exit Game Preview, you need the following items.
  - An approved 1.0 game package that complies with all applicable [Xbox Requirements for Xbox Console Games](/gaming/gdk/_content/gc/policies/console/certification-requirements) or [Xbox Requirements for PC](/gaming/gdk/_content/gc/policies/pc/live-policies-pc), depending on the platforms where your game is released.
  - Approved Store listings that remove Game Preview references and branding.
  - Approved pricing and availability that accurately reflect the release information for the 1.0 version of the game.

Your Microsoft contacts can advise you on the steps to publish the required changes, so that your game is updated at the desired date and time.
