# Game Concepts

[!INCLUDE [reminder](../../includes/managed-creators-only-feature.md)]

Before you can publish an Xbox game that uses full Xbox services, you'll need an approved Game Concept.  

## What is a Game Concept?

Game Concepts give Microsoft an overview of your game, including information such as:
 - Who is developing the game
 - Who is publishing the game
 - The expected release date for Microsoft platforms, and actual release dates if the product is already released on other platforms
 - The platforms where you release or already released the game
 - A description of the game and an overview of the key features
 - Any supplementary material you can provide, including gameplay videos or trailers

## How do I submit a new Game Concept?
Submit all Game Concepts through the online [Xbox Game Submission Form](https://aka.ms/xboxgsf), part of the [Xbox Onboarding Hub](https://www.xbox.com/publish). If you don't have access to the hub or form, contact your Microsoft contacts to get access.  

Submit the Game Concept after you have accurate information that's representative of the final product you'll release. If you need to modify the Game Concept after you submit it, communicate any changes to your Microsoft contacts. If there are significant changes, such as changes to the game features or publishing and distribution plan, you might need to submit a new Game Concept.

### Free-to-play (F2P) games
You can submit a concept for a game in which the base game license is distributed for free and, apart from trials and demos, users can play the base game indefinitely or to its completion, for free. A game that fits this description is considered F2P.

Before submitting a Game Concept for a F2P product, review the F2P game policies and requirements in the Xbox Games Store Policy, available in the [Xbox Publisher Guide](https://aka.ms/xboxpublisherguide).

When submitting a Game Concept for a F2P product, be sure to select the F2P option when completing the Game Submission Form. After selecting the F2P option, provide the required information in the questions that are presented.

## Next steps
After Microsoft reviews your concept, you receive notice of the decision. If Microsoft approves the Game Concept, you receive confirmation of the terms of the concept approval and instructions for proceeding with product creation and setup. If you don't already have a publishing agreement with Microsoft for the game, Microsoft provides one at this stage.

After Microsoft reviews and approves your concept, you're ready to create a new game product in Partner Center and enable full Xbox services.

> [!NOTE]
> Your publishing agreements with Microsoft contain critical information about the requirements that your game is bound to. Ensure you review your publishing agreements in full. Keep a copy of them available for your reference.
 
For more information about product setup, see [Product creation and onboarding](publishing-processes-product-onboarding-and-configuration.md).

If you're planning to release different editions of your game, review the [SKU planning](publishing-processes-sku-planning.md) guidance.
