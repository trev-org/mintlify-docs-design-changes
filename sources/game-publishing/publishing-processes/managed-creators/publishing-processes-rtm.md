# Approve discs for manufacture

[!INCLUDE [reminder](../../includes/managed-creators-only-feature.md)]

Before a disc binary can be Released to Manufacturing (RTM), you must send an approval request to your Microsoft contacts. There are some prerequisites to approval being granted and this topic provides guidance on the information you must provide.

## Requirements for RTM
The following components are required before a disc binary can be approved for RTM.

 1. Certified disc binary
 2. Approved packaging materials

### Certified disc binary

Games that are intended to release on physical discs must have test discs ordered and approved by the Xbox Certification team. If you are Certifying a package that is intended to be manufactured on disc, submit a disc order after successfully submitting the package to Final Certification. Placing a disc order will result in a quantity of test discs being manufactured and delivered to the designated recipients for sign-off and approval to Release to Manufacture (RTM).

For more information, see [Submitting disc and digital games for Final Certification](publishing-processes-disc-and-digital-certification.md).

### Approval of packaging materials

Required packaging artwork must be submitted for review and approval via the Apollo tool in advance of your RTM date. For more information about accessing Apollo, see [Useful Links](../../resources/useful-links.md). For more information about the disc packaging requirements, see [Disc Packaging Requirements](publishing-processes-disc-packaging.md).

In addition, before the pack-out of your product, you must have all other packaging components approved, including any components not approved at the time of RTM. Review the disc packaging templates to ensure that you have provided all required components for review and approval.

#### Related resources

All users of the Apollo tool should read the Apollo User Guide (see [Useful Links](../../resources/useful-links.md) for access), which includes guidance on submitting packaging artwork for review and approval by Microsoft Art Certification. If you have any questions about the Apollo submission process, contact Microsoft Art Certification Support.

For more information, see [Branding and Marketing](../../resources/managed-support/how-to-access-branding-and-marketing-assets.md).

## Royalty tier declarations

Submission of royalty tier declarations is not a requirement to RTM. However, to avoid delays because of manufacturing or billing issues, you should have the royalty tiers selected, submitted, and approved at least 30 days before the start date of manufacture. Royalty tiers must be submitted for all editions (also referred to as *SKUs*) you’re manufacturing, including Review Discs.

> [!NOTE]
> For more guidance about selecting the correct Royalty Tier, use the supplementary information provided in the *Retail Programs and Tier Selection Guide* available in the [Xbox Publisher Guide](https://aka.ms/xboxpublisherguide).

For more information, see [How to submit royalty tiers](../../how-to/how-to-submit-royalty-tiers.md).
