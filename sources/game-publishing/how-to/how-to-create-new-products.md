# How to create new products

[!INCLUDE [reminder](../includes/managed-creators-only-feature.md)]
 
## Definition  

Get started in Partner Center by creating a new product. You can create Game, Game Demo, Game Bundle, and add-on products. Game, Game Demo, and Game Bundle products are created from the Apps and Games overview in Partner Center. Add-on products, including Durables, Consumables, and Add-on Bundles, are created from the add-ons menu within a Game product.

## Products this applies to  
 - Game
 - Game Demo
 - Game bundle
 - Add-on bundle
 - Season Pass&ast;
 - Consumable
 - Durable/Durable with Packages

> [!NOTE]
> &ast;Season Pass products can't be created without Microsoft approval. Speak to your Microsoft representative if you're interested in creating a Season Pass.

## Configuration for Game, Game Demo, and Game bundle

 1. Go to **Partner Center**.
 2. Select the **Apps and Games** workspace.
 3. Select **New product**, and then select the product type you want.
 4. Give your product a name.
 5. **(Game bundles only.)** Select **Create this product in a sandbox**.
 6. If needed, select a **product group**. Product groups are optional and allow you to group related products together in Partner Center.
 7. Select **Create**.
 8. After the product has been created, you can complete each module as required.

## Configuration for add-on products

 1. Go to **Partner Center**.
 2. Go to the product you want to create an add-on product for.
 3. On the product page, select **Add-ons** from the page navigation.
 4. On the add-ons page, select **Create a new** and then select the type of add-on you want to create.
    1. If you're creating a *Season Pass*, select **Add-on bundle**.
    2. For consumable products, Consumable (Microsoft Store-managed) is the common configuration. Speak to your Microsoft representative if you believe Consumable (Developer-managed) is appropriate.
 5. Give the add-on product a unique identifier.
    1. If required (for example, for an add-on bundle product), give the add-on product a **name**. This name must be unique from any other product in the Microsoft Store.
    2. If you're creating a *Season Pass*, select **Make this product a season pass**.
 6. Select **Create**.
 7. After the product has been created, you can complete each module as required.
