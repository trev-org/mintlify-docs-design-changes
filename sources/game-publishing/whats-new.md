# What's new

The Game Publishing Guide (GPG) contains all the information you need to successfully publish a title on the Microsoft Store. This page lists key updates.

## April 2026
 - Added [Packages](concepts/game-package-management.md) with information about the new package management experience in Partner Center.

## February 2026
 - Updated [Technical Capabilities](concepts/properties-technical-capabilities.md) with information about configuring the Xbox Play Anywhere tag displayed in the Microsoft Store.

## November 2025
 
 - Added the [Game Publishing Readiness Checklist](publishing-processes/managed-creators/publishing-processes-game-publishing-readiness-checklist.md), providing recommended reviews before publishing a game to the RETAIL sandbox.
 - Updated [Store listings](concepts/store-listing.md) with the latest guidance for trailer end slate rules and mnemonics.
 - Updated [Branding and Marketing assets](resources/managed-support/how-to-access-branding-and-marketing-assets.md) with the latest asset endpoints.
 - Updated [How to make price changes](how-to/how-to-priceupdate.md) with updated guidance about using automatic currency conversion rates in pricing configurations.
 
## June 2025

 - Refreshed content throughout for Managed Creators.
 - Refreshed guidance for the [Xbox Insider Program](publishing-processes/managed-creators/publishing-processes-xbox-insider-flighting.md).
 - Revamped guidance for publishing a [Game Wish List page](publishing-processes/managed-creators/publishing-processes-wishlist-pages.md).

## Jan 2024

* New [step-by-step guidance](tutorial-xbox-managed/overview.md) for managed partners to publish games to the Microsoft Store on Xbox consoles and Windows PCs.
* Revamped and updated [publishing process](publishing-processes/managed-creators/overview.md) content.
