# Accessibility Feature Tags - Version History

## Version 1.0.1 (3/9/2022)

### Input: No quick-time events - Additional note

The following note was added to the Input: No quick-time events tag:

- Analog sticks and D-pad input related to character or camera movement are exempt from this requirement.
