# item_created

This event is triggered when an item is created.

## Properties

| Name     | Type   | Description                                   |
| :------- | :----- | :-------------------------------------------- |
| ItemId   | String | The string of the item that was created.      |
| ItemType | String | The string type of the item that was created. |
