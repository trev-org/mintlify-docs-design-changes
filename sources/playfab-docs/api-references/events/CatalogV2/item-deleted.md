# item_deleted

This event is triggered when an item is deleted.

## Properties

| Name     | Type   | Description                                   |
| :------- | :----- | :-------------------------------------------- |
| ItemId   | String | The string ID of the item that was deleted.   |
| ItemType | String | The string type of the item that was deleted. |
