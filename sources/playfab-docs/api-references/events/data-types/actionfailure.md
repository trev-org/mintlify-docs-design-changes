# ActionFailure

Information about the error that occurred during action execution.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|FriendlyMessage|String|The friendly message of the action failure.|
|ErrorCode|int32|The error code of the failure.|
