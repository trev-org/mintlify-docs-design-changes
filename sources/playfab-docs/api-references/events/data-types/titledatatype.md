# TitleDataType

Specifies the type of title data.

## Values

|Name|Value|Description|
| :--------------------|:-------------------|:----------------------|
|TitleData|0|Public title data accessible to all clients.|
|TitleInternalData|1|Internal title data only accessible server-side.|
