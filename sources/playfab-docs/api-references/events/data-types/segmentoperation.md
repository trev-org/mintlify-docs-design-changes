# SegmentOperation

The type of segment operation that was performed.

## Values

|Value|Description|
| :--------------------|:----------------------|
|LookUp|A segment lookup operation.|
|Create|A segment creation operation.|
|Delete|A segment deletion operation.|
|Update|A segment update operation.|
