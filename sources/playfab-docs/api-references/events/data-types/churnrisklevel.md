# ChurnRiskLevel

Defines values for the churn risk level of a player.

## Values

|Value|Description|
| :--------------------|:----------------------|
|NoData|Not enough data to make a prediction.|
|LowRisk|The player has a low risk of churning.|
|MediumRisk|The player has a medium risk of churning.|
|HighRisk|The player has a high risk of churning.|
