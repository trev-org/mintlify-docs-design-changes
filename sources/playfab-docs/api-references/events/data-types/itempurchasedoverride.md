# ItemPurchasedOverride

Represents an override used during a purchase event.

## Properties

| Name | Type | Description |
| :--- | :--- | :--- |
| FixedValue | double | The fixed value of the override. |
| ItemId | String | The id of the item. |
| Multiplier | double | The multiplier of the override. |
