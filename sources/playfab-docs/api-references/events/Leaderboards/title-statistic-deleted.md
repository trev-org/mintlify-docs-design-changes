# title_statistic_deleted

This event is triggered when a title statistic is deleted.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|StatisticName|String|Unique name of the statistic.|
|StatisticVersion|uint32|Version of the statistic.|
