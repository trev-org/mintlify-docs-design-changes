# player_verified_contact_email

This event is triggered when a contact email is verified for a player.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|EmailAddress|String|The email address of the player's contact email that was verified.|
|EmailName|String|The name of the player's contact email that was verified.|
|TitleId|String|The ID of the title to which this player event applies.|
