# logged_in_to_title

This event is triggered when a master_player_account entity logs in to a title to obtain a title_player_account entity token.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|EntityTokenId|String|Unique identifier for the generated title_player_account entity token.|
