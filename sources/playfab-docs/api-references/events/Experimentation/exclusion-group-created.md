# exclusion_group_created

This event is triggered when an exclusion group is created.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|Id|String|The unique identifier of the exclusion group.|
|Name|String|Friendly name of the exclusion group.|
|Description|String|Description of the exclusion group.|
