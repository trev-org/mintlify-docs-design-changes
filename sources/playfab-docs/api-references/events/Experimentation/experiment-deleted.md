# experiment_deleted

This event is triggered when an experiment is deleted.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|Id|String|The unique identifier of the experiment.|
|Name|String|The friendly name of the experiment.|
