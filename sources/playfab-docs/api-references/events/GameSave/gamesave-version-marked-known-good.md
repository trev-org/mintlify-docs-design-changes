# gamesave_version_marked_known_good

This event is triggered when a game save version is marked as known good.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|MarkedAt|DateTime|The time at which the game save version was marked as known good.|
|Version|Int64|The version of the game save that was marked as known good.|
