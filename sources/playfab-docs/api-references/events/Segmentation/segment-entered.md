# segment_entered

This event is triggered when a profile enters a segment.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|SegmentId|String|The ID of the segment that had a status change.|
|SegmentName|String|The friendly name of the segment.|
