# title_data_override_deleted

This event is triggered when a title data override is deleted.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|OverrideLabel|String|The override label that was deleted.|
|TitleDataType|[TitleDataType](../data-types/titledatatype.md)|The type of title data.|
