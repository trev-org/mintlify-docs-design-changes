# studio_created

This event is triggered when a studio is created.

## Properties

|Name|Type|Description|
| :--------------------|:-------------------|:----------------------|
|CreatorAuthenticationId|String|Authentication provider's id for the user who created the studio|
|CreatorPlayFabId|String|PlayFab id of the user who created the studio|
|StudioName|String|Name of the new studio|
