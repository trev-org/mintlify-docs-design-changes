# Access policies overview

You can use [Partner Center](https://partner.microsoft.com/dashboard) to allow other services, games, and apps to access your title's Xbox services settings and data.
For example, you might want a web service to display leaderboards on your website, or you might have a companion app that can access the game's title storage to view or modify saved game data.

By default, only the title itself can access the settings and data stored on Xbox services.
You can change this by configuring access policies in Partner Center.
