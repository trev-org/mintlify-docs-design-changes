# Portal configuration overview

Games that are published in [Partner Center](https://partner.microsoft.com/dashboard) and use the full set of Xbox services features can configure their Xbox services in Partner Center.

To access the configuration pages, your game must be enabled for full Xbox services access.

To access the Xbox services configuration pages, after selecting your title in Partner Center, navigate to **Xbox services** > **Gameplay settings**.
