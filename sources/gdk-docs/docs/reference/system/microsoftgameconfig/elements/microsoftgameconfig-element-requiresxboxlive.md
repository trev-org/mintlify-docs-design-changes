# MicrosoftGame.config Element - RequiresXboxLive

Specifies if the title requires Xbox Live connectivity in order to run.

## Parent
This element's parent is the Game element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* Default value is false.
* Allowed values are true and false.

## Remarks
* This element does not control if your title uses Xbox services. This element should only be set if Xbox services connectivity is required for your title to run.
