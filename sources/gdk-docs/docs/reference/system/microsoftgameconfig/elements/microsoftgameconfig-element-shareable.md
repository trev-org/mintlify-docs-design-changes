# MicrosoftGame.config Element - Shareable

Specifies if the Persistent Local Storage is shareable as read-only with other related games. More details about who can read a shareble PLS are found in [Local Storage](../../../../features/console/storage/local-storage.md) 

## Parent
This element's parent is the PersistentLocalStorage element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* Default value is false.
* Allowed values are true or false.
