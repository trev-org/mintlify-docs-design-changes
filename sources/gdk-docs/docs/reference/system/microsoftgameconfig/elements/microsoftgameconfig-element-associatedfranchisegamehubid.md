# MicrosoftGame.config Element - AssociatedFranchiseGameHubId

Specifies the FranchiseGameHubId of the franchise game hub that should be used to launch this hub-aware game.

## Parent
This element's parent is the Game element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* No default value.
* Allowed value is a GUID.
* Applies only to hub-aware games.
