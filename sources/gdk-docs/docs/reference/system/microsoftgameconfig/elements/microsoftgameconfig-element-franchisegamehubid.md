# MicrosoftGame.config Element - FranchiseGameHubId

Specifies the GUID assigned to the franchise game hub, which can be obtained from Partner Center, and will be referenced by all the hub-aware titles. 

## Parent
This element's parent is the Game element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* No default value.
* Allowed value is a GUID.
* Applies only to franchise game hubs.
