# MicrosoftGame.config Element - VirtualMachine

Specifies the title memory allocation on specific Xbox One hardware.

## Parent
This element's parent is the Game element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* No default value.
* Allowed values are child elements of XboxOneXTitleMemory, XboxLockhartTitleMemory, XboxAnacondaTitleMemory, SMTEnabled, RequiresEraShim, XboxCompatibility and DlcCompatibility.
