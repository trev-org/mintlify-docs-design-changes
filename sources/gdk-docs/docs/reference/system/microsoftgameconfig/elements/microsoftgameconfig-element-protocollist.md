# MicrosoftGame.config Element - ProtocolList

Specifies a collection of custom protocols that can be used to launch the game or bring the game into 
the foreground if it were already running.

## Parent
This element's parent is the Game element.

## Requirement
This element is optional and is not required to be set.

## Platform
This element pertains to PC and Console.

## Behavior
* No attributes.
* No default value.

## Remarks
If the game is adding the ProtocolList, they must also add at least one Protocol element as a child.
