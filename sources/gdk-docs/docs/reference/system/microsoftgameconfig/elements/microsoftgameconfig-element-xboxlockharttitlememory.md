# MicrosoftGame.config Element - XboxLockhartTitleMemory

Specifies the title memory setting for Xbox Series S consoles. Allowed values of "Standard" or "Advanced".

## Parent
This element's parent is the VirtualMachine element.

## Requirement
This element is optional and is not required to be set. 

## Platform
This element only pertains to Console devices.

## Behavior
* No attributes.
* Default value is Standard.
* Allowed values are Standard and Advanced.
