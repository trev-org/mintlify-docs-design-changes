# XGameProtocol (deprecated)

> [!IMPORTANT]
> The XGameProtocol APIs are now deprecated. Use the [XGameActivation](../xgameactivation/xgameactivation_members.md) APIs instead. For migration guidance, see [XGameActivation](../xgameactivation/xgameactivation_members.md#remarks).

Reference material for protocol activation.

  
  
## Functions  
  
| Function | Description |  
| --- | --- |  
| [XGameProtocolActivationCallback](functions/xgameprotocolactivationcallback.md) | A callback an application defines for protocol activation events. |  
| [XGameProtocolRegisterForActivation](functions/xgameprotocolregisterforactivation.md) | Registers a callback for protocol activation. |  
| [XGameProtocolUnregisterForActivation](functions/xgameprotocolunregisterforactivation.md) | Unregisters a callback for protocol activation. |
