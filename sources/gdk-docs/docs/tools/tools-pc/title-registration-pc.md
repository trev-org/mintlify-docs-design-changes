# PC game registration

As of the June 2022 GDK, the need to register your game for most scenarios has been removed. If you wish to ensure your icons show up correctly for start menu and desktop tiles (or wish to use these to launch your game), you may do so by explicitly registering using [wdapp.exe register](commandlinetools/gr-wdapp.md#wdapp_register).
