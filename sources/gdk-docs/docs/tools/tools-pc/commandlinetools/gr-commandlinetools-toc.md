# Command-line tools for PC development

Command-line tools used to perform Gaming Runtime&ndash;related tasks on PC.

## Application management tools  

[Application Management (wdapp.exe)](gr-wdapp.md)  
Installs/uninstalls and registers/unregisters applications on the local PC. 

## Manage xgamefiles data on PC

[Managexgamefiles data on PC (xgamesaveutil.exe)](gr-xgamesaveutil.md)  
Manages XGameSave data on PC.

## Screen Capture  

[Captures screenshots on PC (wdcapture.exe)](gr-wdCapture.md)  
Captures screenshots on PC in HDR and SDR formats.

## Video Captures  

[Records live video capture on PC during game tests (wdcapturesession.exe)](gr-wdcapturesession.md)  
Captures live video recording on PC.

## Xbox PC Remote Tools

[Xbox PC Remote listening endpoint (wdEndpoint.exe)](gr-wdEndpoint.md)  
[Xbox PC Remote (wdRemote.exe)](gr-wdRemote.md)

## Packaging tools  

[Make Package (MakePkg.exe)](../../../features/common/packaging/deployment/makepkg.md)  
Creates app packages and app package metadata.

[Make Package Resource Index (MakePRI.exe)](../../../features/common/packaging/deployment/makepri.md)  
MakePRI.exe is used for creating and dumping Package Resource Index (PRI) files and for performing utility functions on them.

[Package Utility (PackageUtil.exe)](../../../features/common/packaging/deployment/packageutil.md)  
Retrieves the signing key information for a package file and compares a base package file with an updated package file.

[Split Package (SplitPkg.exe)](../../../features/common/packaging/deployment/splitpkg.md)  
Builds subset packages from a master package.
