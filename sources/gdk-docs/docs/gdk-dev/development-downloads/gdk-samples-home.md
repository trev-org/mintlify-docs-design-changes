# Microsoft Game Development Kit samples

You can use samples that demonstrate the popular features of the Microsoft Game Development Kit (GDK).

If you need sample requests that aren't available here, share your feedback with us in the [Feedback](https://forums.xboxlive.com/spaces/209/index.html) space on the Xbox Developer Forums. Select the **Give us feedback** button to create your message. 

Because this feedback space is shared with other areas of development, specify that your request is for samples. Be clear and specific in describing the samples you'd like. We prioritize sample requests that receive the most upvotes.

## Download instructions

GDK samples, both console and desktop (Windows PC) samples, are packaged as *a single .zip file*. Download them from [Xbox Developer Downloads](https://aka.ms/gdkdl) site. Unity samples are in a separate .zip file that you can download from the same site.

1. In the dropdown menu under **Select File type**, select **GDK**, and then select **Confirm**.
1. After the page refreshes, in the dropdown menu under **Select build/version number**, select **Microsoft *\<month year>* GDKX (Samples only) compatible with all GDK versions *[(Revision x)]***, where *\<month year>* corresponds to the GDK release you want. The highest revision number indicates the latest update.
1. For Unity samples, select **Microsoft *\<month year>* GDKX Unity Samples**.

An example of the samples file name is **Microsoft March 2023 GDKX (Samples only) compatible with all GDK versions (Revision 2)**.

Alternatively, most of the samples are publicly available on the [GDK Samples GitHub page](https://github.com/microsoft/Xbox-GDK-Samples).

If you're using the Xbox One Software Development Kit, go to [Xbox Developer Downloads](https://aka.ms/xgddl). For more information, see [Microsoft Game Development Kit samples](https://aka.ms/xgdsamples).
