# Step 4 - Integrate the GDK with your game

This section covers initializing the GDK and Xbox/PlayFab services and implementing required features and services. The following steps are required for all games in the Xbox ecosystem.

* [Initializing the GDK with your Game](e2e-services/e2e-initialize-gdk.md)
* [Player sign-in](e2e-services/e2e-user-sign-in.md)
* [Achievements](e2e-services/e2e-achievements.md)
* [Cloud saves](e2e-services/e2e-cloud-saves.md)

## Using game engines with the GDK

The topics in the **Integrate the GDK with your game** section describe using the GDK C/C++ APIs directly to implement the required features and services. Depending on the engine you are using you may need to reference one of the engine guides from the [GDK & game engines](e2e-gdk-and-engines.md) section for engine specific integration details such as using C# wrappers in the Unity Engine or using the online sub-system in Unreal Engine.  

* [Unity Engine](../get-started-with-unity-pc/gdk-unity-end-to-end-guide.md)
* [Unreal Engine](../get-started-with-unreal-pc/gc-get-started-with-unreal-pc.md)
* [Custom C/C++ engine](../get-started-with-custom-engine/gc-get-started-with-custom-engine-pc.md)
