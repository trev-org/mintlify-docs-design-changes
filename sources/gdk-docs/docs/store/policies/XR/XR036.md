# XR-036: In-Title Pricing Information \*

Version 1.0, 3/1/2020

Any real-world currency prices displayed within a title, including but not limited to any promotional or subscription-based discounts, must be sourced from the Xbox catalog.

For more information on in game commerce in the GDK, read the [Commerce section](../../commerce/commerce-nav.md) of the GDK documentation.

## More Information

Titles must use dynamically displayed pricing (that is, they must not show static images displaying prices within a title).
Pricing delivered from Xbox Live will include both the price and currency symbol. Titles should support in-game display of the currency symbol for the user's market.

## Intent

Ensure that pricing information is consistent and trustworthy even in titles that build their own browsing experiences for add-on purchases.
