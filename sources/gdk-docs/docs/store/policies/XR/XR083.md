# XR-083: XDK / GDK Versions

Version 1.2, 07/01/2024

The Microsoft Xbox One Software Development Kit (XDK) and Microsoft Game Development Kit (GDK) game OS that is packaged with a title must be from an approved development kit release.

## More Information
The lists of approved development kit releases and their version information can be found here:

**XDK:** [XDK Approved Libraries](https://aka.ms/xdkapprovedlibs)

**GDK:**  [GDK Approved Libraries](https://aka.ms/gdkapprovedlibs)

## Exemptions

Re-releasing an existing title with a now-expired XDK or GDK is permissible if it is packaged with the same update or QFE as the existing title. Releasing a demo for an existing title with a now-expired XDK or GDK is permissible if it is packaged with the same update or QFE as the existing title.
