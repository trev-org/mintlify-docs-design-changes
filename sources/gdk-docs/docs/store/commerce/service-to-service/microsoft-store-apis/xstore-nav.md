# Microsoft Store Service APIs

This section lists technical reference articles for Microsoft Store service APIs used by game services.
For planning and implementation guidance, see [Manage products from your services](../service-to-service-nav.md).

## In this section  
  
[Fine Grained Rate Limiting for Microsoft Store APIs](../xstore-fgrl.md)  
Describes calling patterns and fine-grained rate limiting for Microsoft Store service calls.  
  
[collections.mp.microsoft.com/v9.0/collections/publisherQuery](xstore-v9-query-for-products.md)  
Describes how to use publisherQuery (v9) to query owned products, including Xbox Game Pass status.  
  
[collections.mp.microsoft.com/v8.0/collections/b2bLicensePreview](xstore-v8-query-for-products.md)  
Describes how to use b2bLicensePreview (v8) to query owned products from your service.  
  
[collections.mp.microsoft.com/v8.0/collections/consume](xstore-v8-consume.md)  
Describes how to fulfill Developer-managed consumables or remove quantity from Store-managed consumables.  
  
[purchase.mp.microsoft.com/v8.0/b2b/orders/query](xstore-v8-clawbackv1.md)  
Describes the legacy query for refunded/revoked consumable purchases.  
  
[purchase.mp.microsoft.com/v8.0/b2b/clawback/sastoken](xstore-v8-clawbackv2-sastoken.md)  
Generates an SAS token to authenticate with your Clawback event message queue.  
  
[purchase.mp.microsoft.com/v7.0/purchases/grant](xstore-v8-grant.md)  
Describes how to grant free app or add-on products to a user from your service.  
  
[purchase.mp.microsoft.com/v8.0/b2b/recurrences/query](xstore-v8-recurrence-query.md)  
Describes how to query a user's subscription products.  
  
[purchase.mp.microsoft.com/v8.0/b2b/recurrences/{recurrenceId}/change](xstore-v8-recurrence-change.md)  
Describes how to change a user's subscription billing state.
