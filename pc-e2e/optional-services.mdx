A GDK title can ship without any of the features below. Pick the ones that fit your design.

## Xbox services

<CardGroup cols={2}>
  <Card title="Achievements" icon="trophy" href="/services/achievements">

  </Card>

  <Card title="Game Saves" icon="floppy-disk" href="/gdk/game-saves">
    Cloud-synced save state via Connected Storage / `XGameSave`.
  </Card>

  <Card title="Social" icon="user-group" href="/services/social">
    Friends, followers, presence.
  </Card>

  <Card title="Rich Presence" icon="signal" href="/services/rich-presence">
    Localized strings describing what the player is doing.
  </Card>

  <Card title="Multiplayer" icon="users" href="/services/multiplayer">
    MPSD sessions, MPM, matchmaking, multiplayer activity.
  </Card>

  <Card title="Player Data" icon="database" href="/services/player-data">
    Stats and leaderboards.
  </Card>

  <Card title="Game Chat 2" icon="microphone" href="/services/game-chat">
    Cross-platform voice chat.
  </Card>

  <Card title="People system" icon="users-rectangle" href="/services/people-system">

  </Card>
</CardGroup>
