## Required software

- [Visual Studio 2022](https://visualstudio.microsoft.com/downloads/)
- [June 2024 GDK](https://github.com/microsoft/GDK/releases/tag/June_2024_Update_2) or later
- [CMake](https://cmake.org/download/) — ensure the installer updates system paths
- [GameMaker Studio 2](https://www.yoyogames.com/)

<Note>
  **Update Gaming Runtime Services (GRTS)** on your dev PC. PC Bootstrapper (May 2022\+) depends on GRTS. Out-of-date GRTS causes sign-in to fail with `E_FAIL`.
</Note>

## GameMaker GDK Extension

The [GameMaker Studio 2 GDK Extension](https://github.com/YoYoGames/GMEXT-GDK) is required to use the GDK with GameMaker.

Clone with submodules:

```bash
git clone --recurse-submodules https://github.com/YoYoGames/GMEXT-GDK
```

See the repo README for build instructions. The solution file is:

```text
<project_root>/source/GDKExtension_gml/extensions/GDKExtension/gdkextension_windows/GDKExtension.sln
```

Build in Visual Studio 2022.

Wiki: [GMEXT-GDK wiki](https://github.com/YoYoGames/GMEXT-GDK/wiki).

## Test the extension

The repo includes a sample at `source/GDKExtension_gml/GDKExtension.yyp`.

In GameMaker: **Build → Create Executable**, then **Package as Zip**.

<Warning>
  **Package as Zip** is required to generate an MSIXVC. **Package as Installer** does not produce an MSIXVC, and the app will not have access to Xbox services.
</Warning>

Install with `wdapp`:

```bat
wdapp install XboxDeveloperExperienceTe.GamePassSampleGame_1.0.0.0_x64__pmsd8ndztetqc.msixvc
```

Launch from the Start menu.

## Configure your MicrosoftGame.config

Sample config: `source/GDKExtension_gml/datafiles/MicrosoftGame.config`.

1. Open Partner Center → your game → **Game setup → Identity details** → **Show Details**.
2. Map values into your `MicrosoftGame.config`:

| Partner Center | MicrosoftGame.config |
| --- | --- |
| Xbox Title ID | `TitleId` |
| Package/Identity/Name | `Identity → Name` |
| Package/Identity/Publisher | `Identity → Publisher` |
| Xbox services → Xbox Settings → MSAAppId | `MSAAppId` |

Example:

```xml
<?xml version="1.0" encoding="utf-8"?>
<Game configVersion="1">
  <Identity Name="41336MicrosoftATG.Achievements2017Redux" Version="1.1.0.0"
            Publisher="CN=A4954634-DF4B-47C7-AB70-D3215D246AF1" />
  <TitleId>64353034</TitleId>
  <MSAAppId>0000000000000000</MSAAppId>
  <ExecutableList>
    <Executable Name="Achievements2017_desktop.exe" TargetDeviceFamily="PC" Id="Game"/>
  </ExecutableList>
  <ShellVisuals DefaultDisplayName="Achievements2017 Desktop Sample"
                PublisherDisplayName="Xbox Advanced Technology Group"
                StoreLogo="Assets\StoreLogo.png"
                Square150x150Logo="Assets\Logo.png"
                Square44x44Logo="Assets\SmallLogo.png"
                Square480x480Logo="Assets\LargeLogo.png"
                Description="Achievements2017"
                ForegroundText="dark"
                BackgroundColor="#000000"
                SplashScreenImage="Assets\SplashScreen.png"/>
</Game>
```

See [MicrosoftGame.config overview](/gdk/game-config).

## Integrate the extension in your project

Minimum integration:

```gml
gdk_init();        // before any other GDK calls
// every frame:
gdk_update();
// on shutdown:
gdk_quit();
```

### Extension modules

| Module | Purpose |
| --- | --- |
| **Base** | User / account selection |
| **Storage** | Per-user load/save (game saves) |
| **Xbox services** | Stats, leaderboards, achievements, rich presence |
| **In-App Purchases** | Microsoft Store commerce via GDK |

API reference: `source/GDKExtension_gml/datafiles/GDK_Extension_Documentation.pdf`.

## Test Xbox services

### Create a test account

See [Creating test accounts](/services/overview).

### Switch your sandbox

1. Find your sandbox ID in **Partner Center → Xbox services → Gameplay settings** (e.g., `ABCDEF.0`).
2. **Start** → **Microsoft GDK Command Prompts**.
3. Run `XblPCSandbox.exe <your sandbox ID>`.
4. Sign in to the Xbox App with your test account.

## Publish

1. Package as Zip in GameMaker (generates MSIXVC).
2. Confirm `MicrosoftGame.config` values match Partner Center.
3. Submit at [Partner Center](https://partner.microsoft.com/dashboard).
