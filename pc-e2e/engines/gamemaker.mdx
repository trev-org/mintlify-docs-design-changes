## Prerequisites

- Visual Studio 2022
- June 2024 GDK or later
- CMake (ensure installer adds to system PATH)
- GameMaker Studio 2 (from [yoyogames.com](https://www.yoyogames.com/))

<Tip>
  Keep **Gaming Runtime Services (GRTS)** and **PC Bootstrapper** up to date. Outdated GRTS causes sign-in failures with `E_FAIL`.
</Tip>

## Set up the GDK Extension

Clone the extension with submodules:

```bash
git clone --recurse-submodules https://github.com/YoYoGames/GMEXT-GDK
```

Follow the README and the [wiki](https://github.com/YoYoGames/GMEXT-GDK/wiki). Build `GDKExtension.sln` in Visual Studio 2022.

## Create a Partner Center product

Follow [Set up an app or game in Partner Center](/publishing/partner-center) before testing Xbox services or publishing.

## Integrate the GDK in your project

Minimum API calls:

- `gdk_init()` â€” call before any GDK functionality
- `gdk_update()` â€” call once per frame
- `gdk_quit()` â€” cleanup at exit

### Modules

| Module | Provides |
| --- | --- |
| **Base** | User/account selection |

## Publish

Before submission:

- MSIXVC package created via **Package as Zip**
- `MicrosoftGame.config` updated with Partner Center values
- GDK extension integrated

Submit via [Partner Center](https://partner.microsoft.com/dashboard/windows/overview).
