**SmartMatch** is Xbox's server-based matchmaking service.

Two API approaches:

- [**Multiplayer Manager (MPM)**](/services/mpm) — wraps SmartMatch with simpler patterns
- **`XblMatchmaking*` functions** — direct API for advanced scenarios

## How SmartMatch works with MPSD

1. Players create a **match ticket session**
2. Request the matchmaking service to find other players
3. A **match ticket** is created
4. The service selects sessions based on config, stats, and request attributes
5. A **match target session** is created
6. Titles run **QoS checks**
7. Gameplay starts

## Hoppers

**Hoppers** are logical buckets where match tickets collect — only tickets in the **same hopper** can match.

### Hopper rules

- **MUST rules** — must be satisfied for tickets to be compatible
- **SHOULD rules** — favored matches

Configured in Partner Center.
