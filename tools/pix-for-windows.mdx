---
title: "PIX for Windows"
description: "PIX for profiling PC GDK titles — DirectX 12, Windows 10+."
---

The PIX shipped in the GXDK profiles **Xbox One development kits**. For PC GDK profiling, install [**PIX for Windows**](https://devblogs.microsoft.com/pix/) separately.

<Warning>
  PIX for Windows runs on **Windows 10 and later** and supports only **DirectX 12**.
</Warning>

PIX for Windows and PIX for Xbox can coexist on the same dev PC.

See the [PIX online documentation](https://devblogs.microsoft.com/pix/documentation/) for installation, capture types, and analysis features.
