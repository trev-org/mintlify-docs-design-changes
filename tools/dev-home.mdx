**Developer Home (Dev Home)** is the on-console developer dashboard — the equivalent of the Home dashboard but on a dev kit or dev-mode retail console. It runs on the console itself and exposes settings, performance counters, and developer-only tools without needing a connected PC.

## Pages in Dev Home

| Page | Use |
| --- | --- |
| **Home** | Pinned launches, recent titles, quick console actions. |
| **Xbox Live** | Sandbox, signed-in accounts, service status. |
| **Settings** | Console name, network, storage, dev mode toggles. |
| **Networking** | Network interfaces, debug NIC, bandwidth tests. |
| **Performance** | Real-time CPU, GPU, memory, and thermal counters. Useful as a sanity check before opening PIX. |
| **Streaming** | Configure the console for game streaming (cloud and remote play). |
| **Report a Problem** | Submit a bug to Microsoft directly from the console, with optional traces. |
| **Manage overlay folders** | Mount and unmount loose-file overlay directories for fast iteration without re-packaging. |

## Overlay folders

Overlay folders are the single biggest productivity feature on a kit — they let your build read fresh assets from a network share without you re-packaging or re-deploying. Configure them in Dev Home or via `xbapp`.
