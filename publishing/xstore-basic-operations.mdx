## Setup: XStoreContext

All `XStore` APIs need an `XStoreContextHandle` from `XStoreCreateContext`.

<Warning>
  On console, **Suspend** or **Quick Resume** invalidates the context. Close and recreate the handle on resume.
</Warning>

## 1. Determine what users can purchase

```cpp
bool CALLBACK ProductEnumerationCallback(const XStoreProduct* product, void* context)
{
    printf("%s %s %u\n", product->title, product->storeId, product->productKind);
    return true;
}

void QueryCatalog()
{
    auto async = new XAsyncBlock{};
    async->queue = m_asyncQueue;
    async->callback = [](XAsyncBlock* async)
    {
        XStoreProductQueryHandle queryHandle = nullptr;
        if (SUCCEEDED(XStoreQueryAssociatedProductsResult(async, &queryHandle)))
        {
            XStoreEnumerateProductsQuery(queryHandle, async->context, ProductEnumerationCallback);
            XStoreCloseProductsQueryHandle(queryHandle);
        }
        delete async;
    };

    XStoreProductKind typeFilter = XStoreProductKind::Consumable
                                 | XStoreProductKind::Durable
                                 | XStoreProductKind::Game;

    XStoreQueryAssociatedProductsAsync(m_xStoreContext, typeFilter, UINT8_MAX, async);
}
```

<Warning>
  Paging is **not optional**. Page sizes vary by environment.
</Warning>

## 2. Evaluate entitlements

Replace `QueryAssociatedProducts` with `QueryEntitledProducts`. Or check `XStoreProduct.isInUserCollection`.

### Durable ownership — license, don't just check ownership

| Scenario | API |
| --- | --- |
| Durable with a package | `XStoreAcquireLicenseForPackageAsync` |
| Durable without a package | `XStoreAcquireLicenseForDurablesAsync` |
| List licensable durables (digital game) | `XStoreQueryAddOnLicensesAsync` |

## 3. Purchase eligible products

```cpp
void MakePurchase(const char* storeId)
{
    auto async = new XAsyncBlock{};
    async->callback = [](XAsyncBlock* async)
    {
        HRESULT hr = XStoreShowPurchaseUIResult(async);
        if (hr == E_GAMESTORE_ALREADY_PURCHASED) {
            printf("Already own this\n");
        }
        delete async;
    };
    XStoreShowPurchaseUIAsync(m_xStoreContext, storeId, nullptr, nullptr, async);
}
```

Provide a way to **refresh ownership on demand** — ideal: initial sign-in and the transition into your in-game store.
