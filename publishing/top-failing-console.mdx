## 001-01 Title Stability

Test on all supported console types. Common failures: crashes/hangs at launch, multiplayer crashes, suspend/resume crashes.

## 003-02 Title Integrity

Must be functionally complete \+ testable with partner services configured. Clean Submission Validator \+ logs.

## 045-01 Respect User Privileges

Don’t let users do things their privileges block. **Always show the system resolution UI** on conflict. Use `XUserCheckPrivilege` \+ `XUserResolvePrivilegeWithUiAsync`.

## 064-02 Joining Game Session from Same Game

Joins/invite-accepts from within the game must enter the multiplayer experience — including password-protected sessions.

## 055-01 Achievements

Most common: criteria met but achievement doesn’t unlock. Verify every unlock.

## 124-01 Game Invitations

Must allow sending invites **from within the game** wherever joinable sessions exist.

## 015-01 User Communication

Check privacy settings. Don’t transmit data or allow communication when not allowed.

## 052-06 Cloud Storage: Roaming

Saves must roam between like devices/platforms when signed in.

## 022-01 Official Naming Standards

Don’t reference competing platforms in text or images.
