---
title: "Choosing the right product type"
description: "Pick from bundles, consumables, durables, and subscriptions for your monetization features."
---

All products are configured in Partner Center.

| Type | When to use | Examples |
| --- | --- | --- |
| Bundles | Multiple products at one price | Season Pass, Game Compilation |
| Consumables | Micro-transactions; repurchaseable | Premium in-game currency |
| Durables | Single-purchase add-ons | Expansion packs, pre-order incentives |
| Subscriptions | Recurring, auto-renew | Publisher or game-specific subscription |

## ProductKind index

| Product type | ProductKind |
| --- | --- |
| Base Game | `Game` |
| Game Bundle | `Game` |
| Add-on Bundle | `Durable` |
| Season Pass | `Durable` |
| Store-managed Consumable | `Consumable` |
| Developer-managed Consumable | `UnmanagedConsumable` |
| Durable with a package | `Durable` |
| Durable without a package | `Durable` |
| Store-managed Subscription | `Pass` |
| Add-on Subscription | `Durable` |

## Bundles

<Warning>
  After a bundle is published to RETAIL, included products cannot be removed.
</Warning>

- **Game Bundle** — includes a base game \+ add-ons or other games
- **Add-on Bundle** — multiple released add-ons under one game
- **Season Pass** — special case of Add-on Bundle that can contain products **not yet released**

## Consumables

| Type | Behavior |
| --- | --- |
| **Store-managed Consumable**_(recommended)_ | Can be purchased repeatedly without fulfillment |
| **Developer-managed Consumable** | Single purchase. Must report fulfillment before user can buy again |

## Durables

| Variant | Use case |
| --- | --- |
| **Durable with a package** | Additional content. License \+ mount required at runtime |
| **Durable without a package** | Extra content in the base game package. License check at runtime |

## Subscriptions

| Feature | Store-managed | Add-on |
| --- | --- | --- |
| ProductKind | `Pass` | `Durable` |
| Visibility | All public storefronts | In-game UI only |
| Pricing models | Per-region custom | Single global price |
| Customer tier switching | Yes | No |
