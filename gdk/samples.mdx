Samples demonstrate the most-used GDK features.

## Where to get them

GDK samples (console \+ desktop) are bundled as a single `.zip` from [Xbox Developer Downloads](https://aka.ms/gdkdl). Unity samples are a separate `.zip` from the same site.

1. **Select File type → GDK → Confirm**
2. **Select build/version number → Microsoft _\<month year\>_ GDKX (Samples only)** — highest revision number = latest
3. For Unity: **Microsoft _\<month year\>_ GDKX Unity Samples**

Most samples are also on the public [Xbox-GDK-Samples GitHub](https://github.com/microsoft/Xbox-GDK-Samples).

Using the Xbox One SDK? See [Xbox Developer Downloads](https://aka.ms/xgddl) and [Xbox One samples](https://aka.ms/xgdsamples).

## Sample categories

- Audio
- Graphics
- IntroGraphics
- Xbox Live
- System
- Tools
- Unity (separate package)
