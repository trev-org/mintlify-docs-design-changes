Porting guides for common migrations to the GDK.

## Engine end-to-end guides

Get started using the GDK with middleware engines: Unity, Unreal, GameMaker.

## Xbox One ERA → GDK (NDA)

Porting an existing Xbox One ERA codebase to the GDK on Xbox. Covers programming model differences, removed WinRT/exceptions, D3D11.x → D3D12.x, and tool changes.

## Memory manager porting (NDA)

Moving your title to the GDK Game OS memory manager. Comparison with the Xbox One ERA memory model and migration guidance.

## FXC → DXC shader porting (NDA)

Migrating HLSL shaders from D3DCompiler (FXC) to DXCompiler (DXC).

## GameInput porting

Moving from `Windows.Xbox.Input` and `XInput` to the unified GameInput API.

## Steam porting (NDA)

For titles already shipped on Steam using Steamworks — mapping Steamworks features to GDK equivalents.
