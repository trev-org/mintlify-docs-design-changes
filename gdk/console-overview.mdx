The Microsoft Game Development Kit (GDK) exposes a set of features that are available **only** when targeting Xbox console hardware — Xbox One family and Xbox Series X|S. These capabilities go beyond what is available on Windows PC and give you direct access to the specialized subsystems built into Xbox consoles: a multi-tier audio pipeline with spatial sound, a high-throughput NVMe-backed storage stack, certified networking APIs, and a system-level video capture and broadcasting service.

If you are building a title that ships on both PC and console, you should isolate these features behind a runtime or compile-time platform check. Attempting to call console-specific APIs on Windows will either fail to compile or return an error at runtime.

## What makes a feature "console-specific"?

A GDK feature is considered console-specific when it relies on hardware or OS services that exist only on Xbox consoles. Practical examples include:

- **Persistent Local Storage (PLS)** — A dedicated, encrypted storage partition that is pre-allocated on the console SSD before your title launches.
- **DirectStorage** — An API that bypasses the CPU for GPU-direct asset decompression, taking advantage of the Xbox Series X|S NVMe controller.
- **Spatial Audio** — Hardware-accelerated Dolby Atmos decoding that routes through the HDMI audio pipeline.
- **XAppCapture** — Deep integration with the system DVR service, which manages clip recording across all titles running on the console.

Windows PC equivalents exist for some of these (for example, DirectStorage on PC), but the console variants have different initialization paths, quotas, and certification requirements.

## Certification requirements

Microsoft enforces Xbox certification requirements (XRs) that apply specifically to console titles. Violating these requirements will cause your submission to fail certification. Key requirements covered in this section include:

| Requirement | Topic |
|---|---|
| XR-133: Local Storage Write Limitations | [Storage](/gdk/storage) |
| Network Security (secure socket usage) | [Networking](/gdk/networking) |
| Game DVR privacy (voice chat exclusion) | [Game DVR](/gdk/game-dvr) |

Review the full [Xbox Certification Requirements](https://learn.microsoft.com/en-us/gaming/gdk/docs/policies/certification/certification-requirements) before submitting your title.
