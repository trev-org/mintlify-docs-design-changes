The **Microsoft Game Development Kit (GDK)** is a complete set of tools, APIs, extensions, and programming models for building games across current and future Microsoft gaming platforms — Xbox consoles, PC (Microsoft Store / PC Game Pass), and Xbox Cloud Gaming.

## What the GDK gives you

- **Continue developing games the way you do today.** Standard Win32 APIs, classic C/C\+\+ patterns, error codes (no exceptions), and familiar async models.
- **Share code across platforms.** One codebase targets PC and Xbox, with minimal platform-specific branching.
- **Fast, reliable iteration.** Improved "F5" deployment reliability and faster launches.
- **Multi-platform services.** Xbox services, Connected Storage, packaging, and accessibility built once and used everywhere.

## What changed from Xbox One ERA

The ERA App Model and WinRT APIs forced studios to maintain Xbox-only code paths. The GDK removed:

- "C\+\+ with hats" extensions
- APIs that throw exceptions
- The WinRT-based async model

In their place: standard Win32 APIs, C-style flat APIs, and a new GDK async pattern aligned with how games are actually written.
