Game streaming brings your game to users on devices without local install.

| Service | Where the game runs | Where the user connects |
| --- | --- | --- |
| **Xbox Cloud Gaming** | Xbox server in an Azure datacenter | Closest datacenter |
| **Xbox Remote Play** | User's home Xbox console | Directly to their console |

## Cloud Aware vs streamed-as-is

Your game **can** stream without changes. But making it **Cloud Aware** lets you optimize for the streaming environment.

[Cloud Aware APIs](/reference/xgamestreaming) let you:

- Detect that the game is currently streaming
- Subscribe to connection state changes
- Adapt UI, input affordances, and resolution to the streaming client

## Private offerings

To test in Azure datacenters before public availability, Microsoft uses **Offerings** — private test environments containing one or more games available to stream.

Contact your Microsoft account representative to customize metadata for streaming or set up a package flight.
