title: "Cloud Aware API calls"
description: "Detect streaming status and react to client connection changes."
---

## Initialize

```cpp
XGameStreamingInitialize();
```

## Check for active streaming clients

```cpp
if (XGameStreamingIsStreaming())
{
    // Game is currently streaming to at least one device.
    // Enable cloud-friendly customizations: lower resolution,
    // touch-friendly UI, latency-tolerant input.
}
```

## React to connection state changes

For live updates when clients connect or disconnect, register with `XGameStreamingRegisterConnectionStateChanged`.
